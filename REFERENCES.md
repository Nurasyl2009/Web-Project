# Пайдаланылған әдебиеттер тізімі (References)

Бұл құжатта "TravelWay" жобасын әзірлеу барысында қолданылған негізгі технологиялар мен дереккөздер тізімі келтірілген.

### 1. Ресми құжаттамалар (Official Documentation)

1.  **React Documentation** — React кітапханасының ресми нұсқаулығы.
    *   URL: [https://react.dev/](https://react.dev/)
2.  **Node.js Documentation** — Node.js серверлік платформасының құжаттамасы.
    *   URL: [https://nodejs.org/en/docs/](https://nodejs.org/en/docs/)
3.  **Express.js Guide** — Веб-фреймворк бойынша нұсқаулық.
    *   URL: [https://expressjs.com/](https://expressjs.com/)
4.  **PostgreSQL Documentation** — Деректер базасын басқару жүйесі бойынша анықтамалық.
    *   URL: [https://www.postgresql.org/docs/](https://www.postgresql.org/docs/)
5.  **Vite Documentation** — Жобаны жинақтаушы құралдың құжаттамасы.
    *   URL: [https://vitejs.dev/guide/](https://vitejs.dev/guide/)
6.  **React Router Documentation** — SPA навигациясын жасау бойынша нұсқаулық.
    *   URL: [https://reactrouter.com/](https://reactrouter.com/)
7.  **Chart.js Documentation** — Интерактивті диаграммалар жасау кітапханасы.
    *   URL: [https://www.chartjs.org/docs/latest/](https://www.chartjs.org/docs/latest/)
8.  **JWT (JSON Web Tokens)** — Аутентификация протоколының сипаттамасы.
    *   URL: [https://jwt.io/introduction/](https://jwt.io/introduction/)

### 2. Оқу ресурстары мен анықтамалықтар (Educational Resources)

9.  **MDN Web Docs (Mozilla Developer Network)** — HTML, CSS және JavaScript бойынша толық анықтамалық.
    *   URL: [https://developer.mozilla.org/](https://developer.mozilla.org/)
10. **W3Schools Online Web Tutorials** — Веб-технологияларды үйренуге арналған платформа.
    *   URL: [https://www.w3schools.com/](https://www.w3schools.com/)
11. **NPM Registry** — JavaScript пакеттерін басқару жүйесі.
    *   URL: [https://www.npmjs.com/](https://www.npmjs.com/) (multer, bcryptjs, pg, cors пакеттері үшін)

### 3. Қолданылған құралдар (Tools)

12. **Visual Studio Code** — Негізгі код редакторы.
13. **Postman / Insomnia** — API сұраныстарын тестілеу құралдары.
14. **Git** — Нұсқаларды басқару жүйесі.
